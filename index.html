1. Boolean Algebra Expressions

Type logic directly using operators. Standard operator precedence applies.

Gate / OperationSupported SymbolsExamples
AND*, &, ANDAB, A(B+C), A * B
OR+, |, ORA + B, A | B
NOT' (suffix), !, ~, NOTA', !A, ~(A+B)
XOR^, XORA ^ B, A XOR B
XNOR=, <->, XNORA = B, A <-> B

2. Canonical Forms

Directly specify truth table rows using minterms or maxterms, with optional don't cares.

FormatSyntaxExamples
Mintermsm(...)m(1,2,3)
MaxtermsM(...)M(0,2,4)
With Don't CaresAdd d(...) or D(...)m(1,3) d(5,7), M(0) D(2)
Custom VariablesPrefix with vars:a,b,c: m(1,2), w,x,y,z: M(0,15)
